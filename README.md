# Personal Portfolio Website

Modern and responsive personal portfolio website built using HTML, CSS, and JavaScript to showcase my skills, projects, certifications, and experience in Software Development, Automation Testing, and SDET.

## 🚀 Features

* Responsive Design for Mobile, Tablet, and Desktop
* Modern UI/UX Design
* Dark/Light Theme Toggle
* Smooth Animations and Transitions
* Projects Showcase Section
* Skills & Technologies Section
* Contact Section
* Fast and Lightweight Performance

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Responsive Web Design
* Git & GitHub
